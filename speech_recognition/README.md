### 전처리
코퍼스를 다운받았을 때, 모든 음성파일들이 mp3포맷을 가지고 있었다.  
실험을 유연하게 진행하기 위하여 Goldwave를 사용하여 모든 음성파일들을 wav포맷으로 바꾸었다.  
포맷을 바꾸면서 모든 파일들은 16,000HZ에 모노타입으로 인코딩되었다.
